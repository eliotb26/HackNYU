About Us

MeCode is a practical tool in the Computer Science learning environment. As touch-screen devices, such as Ipad and Surface Pro, become common tools in learning environment, more and more people are replacing notebooks with these devices. However, these devices are not properly equipt with software that makes writing notes for code easy until now. MeCode allows for the user to take notes and directly transfer these notes to functional code. Using computer vison machine learning API's, MeCode is able to understand the users handwriting while the user is taking notes. This now allows for the personal feel of writing notes, with the ability to test and control your outputs at the same time.

This is not only meant for students, but also can be used to help professors lead courses. Professors can now continue guiding courses with the personal touch other their hand, while still being able to connect with their code. Not to mention, this also eliminates messy handwriting that students cannot read.

So we inventred MeCode, a new way to "write" code.
